== Description ==


### Wordpress Quiz maker

* [Quiz Maker](http://ays-pro.com/index.php/wordpress/quiz-maker)

With our Quiz Maker plugin it’s easy to make a quiz in a short time. Just follow a few simple steps to create online quizzes. What you need to do first is to make questions and then include them into your quiz. You are free to choose the types of the questions. Those are the following: 

* Radio, which is the multiple choice with one right answer.
* Checkbox, which is also the multiple choice but it can include several right answers.
* Text, which requires a longer interpretive answer.
* Dropdown, which is the same multiple choice with one right answer, but it occupies less place.
 
 With Quiz Maker you can also categorize the questions. You can include unlimited number of questions in every quiz.  After passing the quiz, you can see your score. 
Pro version gives you the opportunity to see your mistakes if you have any after completing the quiz.


This plugin allows you make unlimited number of quizes . Each QUIZ can includes unlimited questions. Questions can be single choise, multiple choise, dropdown and text . Plugin is very user friendly and easy to use. 


Don't forget, in case of any problems or upcoming questions feel free to contact us via e-mail info@ays-pro.com.

== Installation ==


There are two ways to install Quiz maker: the easy way, when you install Quiz maker from your WordPress dashboard, and the not so easy way, when you install it from WordPress.org. 

* 1.1 The easiest way to enjoy Quiz Maker:
* 1.1.1	Login to your WordPress dashboard
* 1.1.2	Go to Plugins 
* 1.1.3	Add New
* 1.1.4	Search for Quiz maker
* 1.1.5	Click to install
* 1.2 The second way:
* 1.2.1Download the zip file from https://wordpress.org/plugins/quiz_maker/
* 1.2.2 Go to Plugins 
* 1.2.3 Add New 
* 1.2.4 Upload plugin
* 1.2.5 Choose file ays quiz_maker.zip
* 1.2.6	Click to install

* 1.3 In order to install the Quiz maker from Wordpress.org you must unzip the archive and copy the ays-social-buttons folder into your plugins folder (\wp-content\plugins). 
After one ofthis steps you can activate the plugin from the Plugins menu.
Once activated configure any options as desired and you can enjoy our Quiz maker.
